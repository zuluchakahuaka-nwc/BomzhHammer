Post-Soviet satirical card strategy game. Homeless empires fight for control of dumps, train stations and garbage bins.

## Download & Play

1. Go to [Releases](https://github.com/zuluchakahuaka-nwc/BomzhHammer/releases)
2. Download `BomzhHammer_v0.2-alpha.zip`
3. Unzip anywhere
4. Double-click `Play.bat` — that's it, you're playing

## Controls

| Action | Control |
|--------|---------|
| End Phase / Skip | Space or Enter |
| Main Menu | ESC |
| Play card | Left Click |
| View card info | Right Click |
| Select territory | Left Click |
| Language | Settings button in menu |

## Features

- 32 unit types, 67 situation cards, 9 spells, 5 commanders
- 29 territories on the city map
- 2 ideologies: Alcohol Dictatorship vs Der'mokratiya
- 4 religions: Mnogobomzhie, Trezvost, Alcoteism, Etanolstvo
- AI opponent
- Built-in bot mode (`--bot` flag)
- Languages: Russian / English

## Tech

- Engine: Godot 4.2.2
- Language: GDScript
